Name Type Description
width Number Width of image.

depth Number 1 for gray, 3 for rgb, or 4 for rgba.
diff Number Difference between 2 pixels ranging from 1 to 255. Lower number is more sensitive to change.
percent Number Percent of pixels that meet or exceed difference threshold.

 Function to be called when pixel change is detected.

(static) compareGrayRegions(width, height, regions, buf0, buf1) → {Number}

Get the region specific percentage of different pixels between 2 gray pixel buffers.
Parameters:
Name Type Description
width Number Width of image.
height Number Height of image.
regions Array Array of regions. [{name: "reg1", diff: 5, count: 76800, bitset: Buffer}]
regions[i].name string Name of region.
regions[i].diff Number Difference between 2 pixels, 1 to 255.
regions[i].count Number Total number of pixels in region. (Number of 1's in bitset)
regions[i].bitset Buffer Buffer of 1's and 0's to use as bitset for each pixel of image. 0 means that the index of the pixel is not in the region, while 1 indicates that it is. The pixel array is indexed from top left to bottom right. An image that is 640x480 will have 307200 pixels. The bitset for an image of that size is 307200, with 1's for each pixel that is in the targeted region, and 0 for the rest.
buf0 Buffer Buffer array of pixels.
buf1 Buffer Buffer array of pixels.
